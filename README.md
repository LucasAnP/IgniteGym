### Getting Started

Clone the project repository

```
[https://github.com/LucasAnP/IgniteGym.git](https://github.com/LucasAnP/IgniteGym.git)
```

Access the project directory

```
cd Ignite Gym
```

Access the BACKEND directory 

```
cd backend
npm i
npm start
```

Access the Mobile directory 

```
cd mobile
npm i
npx expo start
```
